Introduction

Web application development involves creating dynamic applications that run on web servers and interact with users through web browsers. This course covers the basics like HTML, CSS, and JavaScript, as well as advanced topics like server-side programming, database integration, and security. You'll learn how to develop and deploy your web applications.

HTML

HTML is the standard markup language for creating web pages.

Example of a basic HTML document:

<!DOCTYPE html> <html> <head> <title>Title</title> </head> <body> <h1>Hello, World!</h1> </body> </html>

CSS

CSS is used to style the HTML elements on web page.

Sample of CSS code:

h1 {   color: blue; }

WSL

WSL allows you to run a Linus distribution alongside your Windows system.

To start a local serveer using WSL:

wsl cs /mnt/c/path/to/your/project python -m http.server

Github

GitHub is an online software development platform. It's used for storing, tracking, and collaborating on software projects. It makes it easy for developers to share code files and collaborate with fellow developers on open-source projects.

Some basic git commands:

  • Copy a repository from GitHub to your local machine.

    • git clone https://github.com/username/repository.git

  • Prepare files to be committed.

    • git add "filename"

  • You can also add all files in the current directory.

    • git add .

  • Save your changes to the local repository.

    • git commit -m "Your commit message"
